# SParaViewer界面说明
## 1.整体说明
![输入图片描述](Markdown%E6%96%87%E4%BB%B6_md_files/%E6%97%A0%E6%A0%87%E9%A2%98-2024-12-13-1540.excalidraw_20260611094116.png?v=1&type=image&token=V1:hTOIFVuyJhAHZ9Lh4PmDU9RfMLjSaU-o5Bzz7eXB7Ms)
软件整体界面参考现代软件风格，例如VScode，简洁、清晰、美观。
总共包含7个部分：Setting Bar、Explore、Plot Area、SNP Port Mapping、TDR Calculate、Other Option、Info Bar。

其中Explore、Plot Area、以及SNP Plot Mapping、TDR Cal、Other Option区域的大小，可以通过拖动区域边界进行控制。

软件支持深色主题与浅色主题，可在Setting中进行设置。
## 2区域功能说明
### 2.1 Setting Bar
Setting Bar区域负责整体软件的设置与一些功能控制
(1)Open:加载一个S参数仿真结果文件，通常以s*p或z*p结尾。
(2)Window：添加一个新的Plot Area
(3)Setting：软件设置，如：主题、字体等
(4)Help：显示帮助信息

### 2.2 Explore
加载S参数仿真文件后，文件会在Explore区域挂载一个文件对应的节点，加载多个文件后，通过点击不同节点进行切换。

### 2.3 Plot Area
Plot Area主要功能是显示S参数波形。具体请参考TouchStone介绍文档。
**X轴**
**频率（Frequency）**，单位通常为 Hz、kHz、MHz 或 GHz
**Y轴**
**S参数的值**，具体取决于显示模式（见下文）
Y轴的三种常见表示方式
1. 幅度（Magnitude）
-   **单位：dB**（最常见）
-   公式：20⋅log⁡10∣Sij∣20⋅log10​∣Sij​∣
-   例如：S11 = -10 dB 表示回波损耗，S21 = -3 dB 表示插入损耗
 2. 相位（Phase）
-   **单位：度（°）或弧度（rad）**
-   表示信号经过网络后的相位偏移
3. 幅度（线性）
-   无单位，直接为 ∣Sij∣∣Sij​∣ 的线性值（0~1 之间）

同时plot需要支持横向缩放、纵向缩放、移动、另存为、Reset等操作。支持直角坐标图与史密斯圆图两种类型。
### 2.4 SNP Port Mapping
该区域是为了优化波形选择流程。例如一个4端口仿真文件(实际中可能为任意数值)，对应了S11~S44一共4*4=16条波形，全部横向或纵向列出的话很很长很丑，这部分应该使用一个方阵进行显示，如：
S11 S12 S13 S14
S21 S22 S23 S24
S31 S32 S33 S34
S41 S42 S43 S44
用户点击对应的位置，Plot Area同步加载对应的Sxx波形，并显示Label。

### 2.5 TDR Calculate
这部分负责TDR的相关计算。配置参数与输出结果请参考14-wp6paper_doshi.pdf文档。
